> 报告日期：2026年3月14日
> 调研人：OpenClaw

## 一、行业概述

### 1.1 行业发展背景

卫星互联网作为未来通信领域的重要方向，具有万亿级市场潜力。我国于2020年将卫星互联网纳入新基建，随后出台多方政策鼓励民营小卫星公司参与国家项目，推动产业发展。

**核心驱动力：**
- 频轨资源争夺激烈，"先到先得"原则下加速布局
- 6G天地一体网络需求
- 地面通信网络的重要补充
- 政策持续加持

### 1.2 市场规模预测

- **2025年**：进入密集发射期，组网建设提速
- **2030年**：市场规模有望突破千亿大关
- **2035年**：GW星座计划完成全年近2500颗卫星发射

> 来源：智研咨询《2025年中国低轨卫星行业产业链全景报告》、方正证券研究

## 三、产业链分析

### 3.1 产业链全景

根据SIA统计，2024年全球卫星产业收入结构：
- 地面设备制造：53.0%
- 运营服务：37.0%
- 卫星制造：6.8%
- 发射服务：3.2%

### 3.2 产业链环节

```
上游：卫星制造 + 火箭发射
    ↓
中游：地面设备制造 + 卫星运营
    ↓
下游：应用服务（宽带通信、物联网、导航增强等）
```

### 3.3 上市公司布局

根据投资者关系互动平台不完全梳理，**2025年11月以来主动披露与卫星产业链关联的公司达95家**。

**A股重点上市公司：**

| 环节 | 公司 | 业务 |
|------|------|------|
| 卫星制造 | 航天智装、航宇微、佳缘科技 | 卫星零部件、测控系统 |
| 地面设备 | 天银机电、上海沪工 | 雷达天线、地面站 |
| 运营服务 | 中国卫通、航天宏图 | 卫星通信、遥感服务 |
| 火箭制造 | 航天动力、航天电子 | 运载火箭配套 |

## 四、头部企业竞争力分析

### 4.1 央国企（国家队）

#### 中国航天科技集团（CASC）
- **优势**：完整产业链、技术积累深厚、发射资源丰富
- **承担任务**：国家重大航天工程、GW星座部分研制
- **旗下机构**：航天五院（卫星总体）、航天一院（火箭总体）

#### 中国航天科工集团（CASIC）
- **优势**：快舟系列火箭、导弹技术转民用
- **特点**：专注于商业发射服务

#### 中国星网集团
- **定位**：GW星座统筹建设主体
- **优势**：政策支持、资源整合能力
- **挑战**：商业化运营经验

## 五、竞争力对比分析

### 5.1 央国企 vs 民企

| 维度 | 央国企 | 民企 |
|------|--------|------|
| **技术积累** | 深厚，长期积淀 | 灵活，创新快 |
| **资源获取** | 发射场、频轨资源 | 需竞争获取 |
| **资金实力** | 国家投入 | 依赖融资 |
| **商业化效率** | 较低 | 较高 |
| **成本控制** | 较高 | 较低 |
| **典型优势** | 重大工程、可靠性 | 快速迭代、成本效益 |

### 5.2 发展模式

**国家队模式：** "顶层设计+市场参与"
- 中国星网统筹GW星座
- 三大运营商（移动/联通/电信）成为连接卫星与用户的关键纽带

**民企模式：** "特斯拉式"
- 车企跨界（时空道宇）
- 快速融资、迭代（银河航天）
- 差异化竞争（遥感、特种卫星）

### 5.3 2025-2026年关键节点

| 时间 | 事件 |
|------|------|
| 2025下半年 | 多个星座进入批量发射阶段 |
| 2026 | GW星座大规模组网发射启动 |
| 2026 | 民营火箭集中首飞/入轨 |
| 2026 | 多家民营火箭/卫星企业IPO |

## 六、结论与展望

### 6.1 主要结论

1. **双核心格局形成**：GW星座（国家队）+ 千帆星座（商业）构成中国卫星互联网双核心

2. **民营力量崛起**：2025年在轨卫星中民营占比达60%，民营火箭密集首飞

3. **产业链逐步完善**：A股95家上市公司布局，覆盖制造、发射、运营、服务全链条

4. **资本热度高**：2025年商业航天融资近100亿元，IPO窗口开启

5. **差距与挑战**：
   - 火箭运力仍不足（相较SpaceX）
   - 卫星批量制造能力待提升
   - 商业应用场景仍需拓展

### 6.2 未来展望

- **2028年**：GW星座预计达成"百箭千星"
- **2030年**：市场规模破千亿
- **2035年**：完成1.3万+1.5万颗卫星部署，构建6G天地一体网络
