# A股中长期优质股票推荐报告

**报告日期：2026年3月12日**

---

## 一、核心观点

根据2026年3月各大机构研报，A股中长期应聚焦以下四大主线：
1. **AI产业链**（光模块、半导体、机器人）
2. **电力出海/新能源**
3. **资源周期品**（铜、铝、油气）
4. **非银金融**

> 数据来源：新浪财经《十大机构看后市》(2026-03-08)
> https://finance.sina.com.cn/stock/marketresearch/2026-03-08/doc-inhqhism4377086.shtml

## 四、风险提示

- 地缘冲突导致的市场波动
- 美联储货币政策不确定性
- AI算力需求增速不及预期
- 光模块价格年均降幅15%-20%
