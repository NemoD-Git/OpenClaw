## 三、重要投资观点摘录

### AI 投资逻辑演进
| 时间 | 表述 |
|------|------|
| 2023年 | "AI是个大方向，值得关注，但目前还没看懂，不会轻易下手" |
| 2024年 | "AI是工业革命级别的变革，影响可能超过互联网与传统工业革命，不能错过" |
| 2025年Q3 | 首次建仓英伟达（0.7%），理由：佩服黄仁勋"十年讲同一件事"的战略定力 |
| 2025年Q4 | 11倍加仓英伟达，建仓三家AI细分龙头 |
| 2026年1月 | "新的一年，需要认真学习怎么使用AI，希望理解到敢下重注的地步" |
| 2026年 | "这玩意儿真是越用越觉得有意思，感觉世界有点不一样了" |

### 苹果观点
- 2011年买入时列五大理由：极致用户体验、成熟平台生态、单一产品模式、高效营销、巨大市场潜力
- "苹果生意模式比可口可乐更好，用户忠诚度高，提价后用户仍会购买"
- "苹果市盈率还有31倍，25倍以下会感觉好很多"（2025年4月）

### 茅台观点
- "180的茅台已经很便宜，有钱就先买了"
- "茅台价格下降不等于公司状况不好，酒其实还是买不到"
- "i茅台真能再造两个茅台"

### 腾讯观点
- 持续看好微信生态
- "腾讯价格合适时愿意加仓"
