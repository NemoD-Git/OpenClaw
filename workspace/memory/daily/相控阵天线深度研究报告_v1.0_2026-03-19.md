## 摘要

相控阵天线技术作为现代无线通信与雷达系统的核心支撑技术，近年来在卫星互联网、机载平台、车载终端及舰载系统等领域取得了显著进展。本报告聚焦全球相控阵天线技术的前沿发展动态，重点分析Starlink等顶流厂商的产品技术参数，评估国内主要供应商的技术水平，并从技术参数、成本、可靠性等多维度进行综合分析，为相关领域的研究人员与工程技术人员提供参考。

## 第一章 研究背景与研究方法

### 1.1 研究背景

相控阵天线（Phased Array Antenna）是一种通过电子方式控制天线波束指向的阵列天线系统，无需机械转动即可实现快速波束扫描。自20世纪60年代军事雷达应用起步以来，相控阵天线技术已从军用领域逐步扩展至民用通信、卫星互联网、自动驾驶等多元化应用场景。

进入2025年，随着低轨卫星互联网的快速部署（以SpaceX Starlink为代表）、5G毫米波基站的大规模建设、以及6G技术研发的推进，相控阵天线技术迎来了前所未有的发展机遇。根据市场研究机构预测，2026年全球相控阵天线市场规模将达到56-63亿美元，预计2035年增长至150-170亿美元，复合年增长率约为11.49%。

## 第二章 相控阵天线技术基础与关键参数体系

### 2.1 相控阵天线工作原理

相控阵天线通过控制天线阵列中各辐射单元的信号相位，实现波束的电子扫描。其核心原理可概括为：

1. **阵列几何配置**：天线单元按一定规则排列（线阵、面阵、圆柱阵等）
2. **相位控制**：通过移相器改变各单元信号的相对相位
3. **波束形成**：多路信号在空间矢量叠加，形成定向波束
4. **波束扫描**：改变相位梯度实现波束指向变化

相控阵天线相比传统机械扫描天线的核心优势包括：

| 特性 | 相控阵天线 | 传统机械天线 |
|------|------------|--------------|
| 扫描速度 | 微秒~毫秒级 | 秒级 |
| 可靠性 | 无机械磨损部件 | 机械部件易损 |
| 多波束 | 支持同时多波束 | 通常单波束 |
| 适应性 | 捷变能力强 | 响应慢 |

### 2.2 核心性能参数定义与测试方法

#### 2.2.1 等效全向辐射功率（EIRP）

EIRP（Equivalent Isotropically Radiated Power）是表征天线系统辐射能力的核心指标，定义为：

```
EIRP = Gt × Pt
```

其中Gt为发射天线增益（dBi），Pt为发射功率（dBW或W）。

**Starlink卫星EIRP数据：**

| 参数 | 数值 | 说明 |
|------|------|------|
| 单星最大EIRP | **4.89 MW** | 下行链路 |
| 等效自由空间衰减 | **~125 dB** | 地球同步轨道高度 |
| 终端接收功率密度 | **~15 nW** | 典型用户终端 |

#### 2.2.2 接收品质因数（G/T）

G/T值是接收系统的品质因数，表征接收灵敏度：

```
G/T = G - 10×log10(T)
```

其中G为接收天线增益（dBi），T为系统噪声温度（K）。

**Starlink终端G/T值：**

| 参数 | 数值 | 来源 |
|------|------|------|
| G/T范围 | **见FCC Schedule S** | FCC技术报告 |
| 测试频段 | **Ku波段（10.7-12.7 GHz）** | FCC文件 |
| 备注 | 包含LNA噪声贡献 | 技术分析 |

#### 2.2.3 轴比（Axial Ratio）

轴比是衡量圆极化天线极化纯度的重要指标，定义为：

```
AR = 20×log10(E_max/E_min) [dB]
```

理想圆极化轴比为0dB（无限大比值），工程应用中通常要求：

| 应用场景 | 轴比要求 |
|----------|----------|
| 卫星通信 | ≤3 dB（优）/≤6 dB（良） |
| GPS导航 | ≤1.5 dB |
| 雷达极化测量 | ≤0.5 dB |

#### 2.2.4 扫描角（Scan Angle）

扫描角指波束偏离天线法线的最大角度，是相控阵天线的重要设计指标。

| 阵列类型 | 最大扫描角 | 限制因素 |
|----------|------------|----------|
| 普通面阵 | ±60°（方位/俯仰） | 栅瓣、增益下降 |
| 宽角扫描阵 | ±75° | 阵列设计复杂度 |
| 共形阵 | 全空域 | 馈电网络设计 |

**典型AESA雷达扫描参数：**

| 参数 | 数值 | 说明 |
|------|------|------|
| 方位扫描角 | **±60°** | 典型军用雷达 |
| 俯仰扫描角 | **±45°** | 典型军用雷达 |
| 扫描速度 | **<100 μs** | 波束切换时间 |

## 第八章 研究结论与展望

### 8.1 主要研究结论

1. **Starlink技术领先**：在低轨卫星互联网领域，SpaceX的相控阵天线技术代表了当前消费级最高水平，其110°波束覆盖、轻量化设计（2.9-3.2kg）、宽温度适应（-30°C to 55°C）等参数处于行业领先地位。

2. **机载应用前景广阔**：Starlink Aviation已完成30,000+架次飞行验证，下载速率40-220 Mbps，延迟20-40 ms，性能显著优于传统GEO卫星方案。

3. **国内技术快速追赶**：以CETC/55所、国博电子为代表的国内厂商在军用相控阵领域已达到国际先进水平，但在商业航天、消费级应用方面仍有差距。

4. **成本持续下降**：相控阵天线成本随规模化呈现显著下降趋势，预计2030年机载终端价格将降至$50,000-80,000区间。

### 8.2 研究局限性

1. 部分厂商关键参数（如EIRP、G/T值）未公开披露，数据存在不完整性
2. 市场数据来源多为商业研究报告，存在统计口径差异
3. 技术参数时效性限制，最新数据可能存在滞后
