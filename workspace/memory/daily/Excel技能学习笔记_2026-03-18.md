## 一、数据透视表（徐军泰）

### 1.1 什么是数据透视表

数据透视表是 Excel 中**最强大的数据分析工具**，适合：
- 数据汇总
- 统计分析
- 多维度交叉分析
- 快速生成报表

### 1.2 创建数据透视表

**方法一：快捷键**
1. 选中数据区域任意单元格
2. 按 `Alt + N + V`，再按 `V`（或直接按 `Alt + D + P`）
3. 选择数据透视表放置位置（新工作表/现有工作表）
4. 点击"确定"

**方法二：菜单操作**
1. 选中数据区域任意单元格
2. 点击菜单：**插入** → **数据透视表**
3. 选择放置位置
4. 点击"确定"

### 1.3 数据透视表四大区域

| 区域 | 作用 | 说明 |
|------|------|------|
| **行** | 纵向展示数据 | 相当于分类标签 |
| **列** | 横向展示数据 | 可添加多个字段 |
| **值** | 数值计算区域 | 求和、计数、平均值等 |
| **筛选** | 数据筛选器 | 可按字段快速筛选 |

### 1.4 字段列表操作

1. 在右侧字段列表中**勾选**字段，该字段自动添加到透视表
2. 拖动字段到不同区域，透视表实时更新
3. 右键字段可设置：
   - 字段设置（自定义名称）
   - 值显示方式（总计百分比、排名等）

### 1.5 切片器（Slicer）

切片器是数据透视表的**可视化筛选器**，操作步骤：

1. 点击数据透视表任意单元格
2. 点击菜单：**分析** → **插入切片器**
3. 选择需要筛选的字段（可多选）
4. 点击"确定"
5. 在切片器中点击即可筛选数据

**切片器高级操作：**
- 多个切片器可**联动**
- 右键切片器 → **报表连接** → 选择关联的数据透视表
- 调整切片器显示顺序：右键 → **上移/下移**

### 1.6 值的汇总方式

| 汇总方式 | 说明 |
|----------|------|
| 求和 | 数值求和（默认） |
| 计数 | 计数（非空单元格） |
| 平均值 | 计算平均值 |
| 最大值/最小值 | 极值 |
| 产品 | 数值乘积 |

**修改方法：** 点击值区域字段 → **值字段设置** → 选择汇总方式

### 1.7 分组功能

1. 右键行/列字段
2. 选择**创建组**
3. 设置分组条件（如日期分组：年/季度/月份）

### 1.8 计算字段

添加自定义计算列：

1. 点击数据透视表
2. **分析** → **字段、项目和集** → **计算字段**
3. 输入名称和公式
4. 例如：`=数量*单价`

### 1.9 数据透视表刷新

当数据源更新时：

1. 右键数据透视表 → **刷新**
2. 或点击 **分析** → **刷新**
3. 全部刷新：**分析** → **刷新全部**

## 二、VLOOKUP 函数（星星米葵花）

### 2.1 函数语法

```
=VLOOKUP(查找值, 查找区域, 返回列数, 匹配方式)
```

| 参数 | 说明 | 示例 |
|------|------|------|
| 查找值 | 要查找的内容 | A2, "张三" |
| 查找区域 | 包含数据的范围 | A:D |
| 返回列数 | 返回第几列 | 2 |
| 匹配方式 | 0=精确, 1=模糊 | 0 |

### 2.2 基础精确查找

**场景：** 根据工号查找员工姓名

```
=VLOOKUP(E2, A:C, 2, 0)
```

**步骤：**
1. 在目标单元格输入公式
2. 第一个参数：查找值（工号）
3. 第二个参数：数据区域（A:C）
4. 第三个参数：返回第2列（姓名）
5. 第四个参数：0（精确匹配）

### 2.3 模糊匹配（区间查找）

**场景：** 根据分数划分等级

```
=VLOOKUP(B2, $E$2:$F$5, 2, 1)
```

**要求：** 查找区域第一列必须**升序排列**

| 分数区间 | 等级 |
|----------|------|
| 0-60 | 不及格 |
| 60-70 | 及格 |
| 70-80 | 良好 |
| 80-100 | 优秀 |

### 2.4 逆向查找（从右向左）

VLOOKUP 只能从左向右查找，逆向查找需要技巧：

**方法：IF({1,0}, ...) 构建虚拟数组**

```
=VLOOKUP(E2, IF({1,0}, B:B, A:A), 2, 0)
```

**原理：** IF({1,0}, B列, A列) 将B列放到第一列，A列放到第二列

### 2.5 多条件查找

**场景：** 同时按"姓名"和"城市"查找销售额

```
=VLOOKUP(E2&F2, IF({1,0}, A:A&B:B, C:C), 2, 0)
```

**注意：** 需要按 **Ctrl + Shift + Enter** 三键确认（数组公式）

### 2.6 通配符查找

**场景：** 查找以某字符开头的姓名

```
=VLOOKUP("张*", A:B, 2, 0)
```

- `*` 代表任意字符
- `?` 代表单个字符

### 2.7 近似匹配查找

**场景：** 查找最接近但不超过的值

```
=VLOOKUP(65, A:B, 2, 1)
```

前提：查找区域第一列升序排列

### 2.8 跨表查找

```
=VLOOKUP(A2, Sheet2!A:B, 2, 0)
```

引用其他工作表的数据

### 2.9 跨文件查找

```
=VLOOKUP(A2, '[工资表.xlsx]Sheet1'!A:B, 2, 0)
```

### 2.10 查找不到时显示特定值

```
=IFERROR(VLOOKUP(A2, B:C, 2, 0), "未找到")
```

### 2.11 返回多列数据

在多列中分别使用 VLOOKUP：

```
=VLOOKUP($E2, $A:$C, 1, 0)  // 第1列
=VLOOKUP($E2, $A:$C, 2, 0)  // 第2列
=VLOOKUP($E2, $A:$C, 3, 0)  // 第3列
```

### 2.12 批量查找

结合 INDEX + MATCH 更灵活：

```
=INDEX(B:B, MATCH(E2, A:A, 0))
```

### 2.13 动态列数查找

使用 COLUMN 函数：

```
=VLOOKUP($A2, $E:$G, COLUMN(B1), 0)
```

向下填充时自动切换返回列

### 2.14 处理空格问题

结合 TRIM 函数：

```
=VLOOKUP(TRIM(A2), B:C, 2, 0)
```

### 2.15 处理数据类型不一致

使用 TEXT 或 VALUE 转换：

```
=VLOOKUP(A2&"", B:C, 2, 0)  // 数值转文本
=VLOOKUP(A2*1, B:C, 2, 0)    // 文本转数值
```

### 2.16 返回第N次出现的值

使用辅助列 + COUNTIF：

```
=VLOOKUP($E2&COUNTIF($E$2:E2,E2), A:B, 2, 0)
```

### 2.17 高亮显示查找到的数据

条件格式：
1. 选中数据区域
2. **开始** → **条件格式** → **新建规则**
3. 使用公式确定格式：
   ```
   =COUNTIF($E:$E, $A1)>0
   ```
4. 设置填充颜色

### 2.18 VLOOKUP 与 XLOOKUP（新版）

新版 Excel 推荐 XLOOKUP：

```
=XLOOKUP(查找值, 查找列, 返回列, 未找到值)
```

**优势：**
- 支持逆向查找
- 默认精确匹配
- 更简洁

## 三、快速填充/智能填充（高薪表匠）

### 3.1 什么是快速填充

快速填充（Flash Fill）是 Excel 的**智能数据处理功能**，能自动识别数据规律并批量填充。

### 3.2 基本操作方法

**方法一：快捷键**
1. 在第一行输入期望的结果
2. 按 `Ctrl + E`
3. Excel 自动识别规律并填充

**方法二：菜单操作**
1. 输入第一个示例
2. 点击 **数据** → **快速填充**

**方法三：自动提示**
1. 输入第一个示例
2. 当 Excel 出现智能提示时，按 `Enter` 确认

操作：

### 3.9 与函数的对比

| 场景 | 快速填充 | 函数 |
|------|----------|------|
| 简单提取 | ✅ 一键完成 | 需要 LEFT/RIGHT/MID |
| 合并 | ✅ 一键完成 | 需要 & 或 CONCATENATE |
| 复杂规律 | ❌ 可能失败 | ✅ 灵活 |
| 数据更新 | ❌ 需重新填充 | ✅ 自动更新 |

### 3.10 注意事项

1. **需要至少一个示例**：Excel 需要示例来识别规律
2. **数据需连续**：最好在同一列
3. **规律需一致**：示例应代表典型情况
4. **可撤销**：按 `Ctrl + Z` 撤销

### 3.11 高级技巧

**技巧1：处理不规则数据**
- 输入多个示例让 Excel 更好识别
- 如果一次失败，可手动输入2-3个示例

**技巧2：结合函数使用**
- 先用函数处理部分数据
- 再用快速填充处理剩余

**技巧3：处理特殊情况**
- 如果 Excel 识别错误，手动输入更多正确示例
