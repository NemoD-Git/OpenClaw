## 一、行业概述

### 1.1 相控阵天线在卫星通信中的核心地位

相控阵天线是卫星互联网体系中的关键设备，决定整张网络的长期运行能力。在卫星互联网体系中，相控阵天线正从"高端通信设备"转变为"决定整张网络是否能长期运行的关键能力"。

**相控阵天线核心原理：**
- 通过控制阵列天线中各单元的相位和幅度，实现波束的电子扫描
- 无需机械转动即可实现快速波束指向
- 可实现多波束、跳波束等高级功能

**技术分类：**

| 类型 | 特点 | 应用场景 |
|------|------|---------|
| 无源相控阵（PESA） | 集中式发射/接收，TR组件较少 | 早期雷达 |
| 有源相控阵（AESA） | 每个单元独立TR组件，高可靠性 | 主流卫星通信 |
| 数字相控阵 | 数字波束形成，灵活度高 | 下一代系统 |

## 三、中国市场现状

### 3.1 产业链结构

```
上游：芯片/元器件（TR芯片、功率放大器、移相器、滤波器）
    ↓
中游：T/R组件 → 相控阵天线模块 → 整机系统
    ↓
下游：卫星载荷（星载） + 地面终端（车载/船载/机载）
```

### 3.2 核心环节价值分布

**卫星系统成本结构：**
- 载荷：占比75%
- 平台：占比25%

**载荷系统中：**
- 相控阵天线：占比75%
- 其中TR组件：占比25%

## 九、结论与展望

### 9.1 主要结论

1. **技术路线多元化**：GaAs/GaN/CMOS/液晶多条技术路线并存
2. **产业链完善**：形成从芯片到整机的完整体系
3. **央国企主导**：中电科13所/55所、航天五院为核心
4. **民企崛起**：铖昌科技、国博电子等在细分领域突破
5. **应用场景扩展**：从星载向车载、船载、机载延伸

### 9.2 发展趋势

- **高频化**：向毫米波、Ka/Ku波段发展
- **低成本化**：CMOS、液晶新技术路线
- **智能化**：AI波束管理、快速寻星
- **国产化**：核心芯片自主可控
- **集成化**：从TR组件到微系统一体化

### 9.3 挑战与机遇

**挑战：**
- 高端TR芯片仍依赖进口
- 量产工艺与成本控制
- 终端市场培育

**机遇：**
- 政策持续支持
- 卫星互联网组网加速
- 5G/6G融合需求
